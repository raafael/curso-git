Curso de Git

Trabalhando em Equipe com Controle e Segurança para seus projetos

Este curso explora os benefícios de utilizar o Git como ferramenta de controle de versão para projetos em qualquer linguagem, em qualquer plataforma.

Principais benefícios:

  • Funciona de maneira distribuída
  • Permite a edição concorrente de arquivos do projeto
  • Auto Merge
  • Não necessita conexão, trabalho é salvo localmente para posterior commit
